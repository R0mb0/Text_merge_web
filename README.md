<h2>üöÄ Features</h2>
<ul>
<li><strong>100% Privacy-First</strong>: Everything happens locally in your browser. No files are ever uploaded to a server, ensuring your data remains completely private.</li>
<li><strong>Zero Dependencies</strong>: No frameworks or heavy libraries. Built with pure, lightweight Vanilla JavaScript, HTML5, and CSS3.</li>
<li><strong>Smart Formatting</strong>: Toggle the option to automatically include the original file names as clear, visual headers in the final merged document.</li>
<li><strong>Custom Output</strong>: Automatically detects the predominant file extension of your uploads, while letting you customize the final output format.</li>
<li><strong>Drag & Drop Interface</strong>: Intuitive and smooth file management. Easily add multiple files at once or drop them directly into the designated area.</li>
<li><strong>Modern UI</strong>: Fast, responsive, and adaptive Dark/Light mode interface based on your system preferences, complete with a progress bar for visual feedback.</li>
</ul>

<h2>üõ†Ô∏è How it works</h2>
<ol>
<li><strong>Drag and drop</strong> your text files into the dropzone.</li>
<li>Manage your list by removing any unwanted files.</li>
<li>Select whether you want to include original file titles as separators and set your desired output extension.</li>
<li>The app asynchronously reads your files using the native <code>File.text()</code> API to prevent UI freezing.</li>
<li>It concatenates the content and creates a local <code>Blob</code>.</li>
<li>Your browser instantly prompts you to download the merged file!</li>
</ol>

<h2>üèÜ What makes it special?</h2>
<ul>
<li><strong>Instant Offline Access</strong>: Since it doesn't require a backend, you can literally save the <code>index.html</code> file on your desktop and use it offline forever.</li>
<li><strong>No Build Step</strong>: You don't need Webpack, Vite, or npm to run or modify this project. Just open the code in any text editor.</li>
</ul>
